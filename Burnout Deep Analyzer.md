## GOAL

Analyze structured performance logs over a defined period to:

1. Detect burnout and sustainability risks.
2. Track trends over time and create longitudinal snapshots.
3. Forecast future risk levels based on observed patterns.
4. Suggest mitigation strategies.
5. Assist with professional communication to supervisors if needed.

# HOW TO USE

1. Run monthly or quarterly.
2. Paste structured daily logs for the period.
3. Include optional self-assessed stress or job satisfaction (1–10).
4. If fewer than 10 entries, provide low confidence warning.
5. If fewer than 5 entries, limit insights with explicit notice.
6. Append the output to your master markdown file.
