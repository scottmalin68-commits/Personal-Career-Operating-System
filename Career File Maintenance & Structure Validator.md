# Career File Maintenance & Structure Validator
Author: Scott M
Version: 1.2
Last Updated: 2026-02-11

## GOAL

Validate, repair structure, and maintain consistency of markdown files generated by the Career Operating System:

1. Ensure required sections exist and are correctly ordered
2. Fix minor formatting (headers, bullets, tables, indentation)
3. Flag/fill missing metadata with placeholders
4. Support four main files:
   - daily_logs.md
   - aggregated_activity_summary.md
   - promotion_assessment.md
   - burnout_trends.md
5. Append optional timestamped maintenance note
6. Produce concise maintenance report

Only structural/metadata changes — never alter user content.

## MAINTENANCE RULES

1. Header & Order — Enforce the templates above. Move/add missing sections if needed.
2. Formatting — Use consistent bullets (-), tables with aligned pipes, code blocks with ```, indentation 2 or 4 spaces.
3. Metadata — Required fields: Author, Version, Last Updated, Period (if applicable).
   Missing values → insert placeholders:
   - Version: 1.0
   - Last Updated: YYYY-MM-DD
   - Period: Unknown
   - Author: Scott M (if blank)
4. Content — Never delete, rewrite, or add meaning — only fix structure/formatting.
5. Maintenance Note (optional) — Append at bottom:
   <!-- Maintained by AI on YYYY-MM-DD – structural fixes only -->
